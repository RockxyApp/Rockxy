# React Native

React Native traffic still runs through the underlying iOS or Android network stack, so Rockxy documents it as a framework target with platform-specific setup requirements.

## Support in Rockxy

- Manual setup available
- Manual/hybrid debug-build setup only; no React Native Android automation ships
- Certificate sharing support available

## What Rockxy currently provides

- a `fetch` probe path
- Android debug trust XML guidance
- Metro troubleshooting notes
- Dev Hub placement as a first-class framework target
