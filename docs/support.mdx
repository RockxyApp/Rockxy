Rockxy is independently maintained as an open-source, local-first macOS debugging proxy. Financial support helps sustain development, release infrastructure, documentation, security work, and other expenses that directly support the project.

## Fund Rockxy on Open Collective

<Card title="Contribute on Open Collective" icon="heart" href="https://opencollective.com/rockxy/donate">
  Make a one-time or recurring contribution and view Rockxy's public budget, contributors, transactions, and expenses.
</Card>

Rockxy is fiscally hosted by [Open Source Collective](https://docs.oscollective.org/welcome-and-introduction-to-osc/what-is-fiscal-hosting). The fiscal host receives and manages project funds, reviews eligible expenses, and provides financial and legal infrastructure. Open Collective provides the public fundraising and ledger platform.

This structure keeps project finances visible to the community. Open Source Collective currently charges a 10% host fee on incoming funds, and payment processors may charge additional fees. Contributions to projects hosted by Open Source Collective are not tax-deductible. See the fiscal host's current [fee schedule](https://docs.oscollective.org/how-it-works/fees) and [fiscal-hosting explanation](https://docs.oscollective.org/welcome-and-introduction-to-osc/what-is-fiscal-hosting) before contributing.

## Other Ways to Help

Financial support is only one way to move Rockxy forward:

- Star the [GitHub repository](https://github.com/RockxyApp/Rockxy) and share it with developers who debug macOS, mobile, API, WebSocket, GraphQL, model API, or Web3 traffic.
- Report reproducible bugs and propose focused features through [GitHub Issues](https://github.com/RockxyApp/Rockxy/issues).
- Help other users and share workflows in [GitHub Discussions](https://github.com/RockxyApp/Rockxy/discussions).
- Improve tests, documentation, accessibility, localization, or code by following the [contribution guide](https://github.com/RockxyApp/Rockxy/blob/main/CONTRIBUTING.md).

Funding does not change Rockxy's commitment to a complete, local-first Community app. Project priorities remain grounded in user value, maintainability, security, and the public roadmap rather than sponsor-only control.
